# 中国各省份GDP数据可视化系统

## 项目概述

本项目是基于Python的大数据可视化实践作业，使用**真实的中国各省份GDP数据**，结合SQLite数据库和多种可视化技术，实现了一个功能完整的GUI数据可视化系统。

## 三大特性

### 1. 科学性
- 使用**真实的中国各省份GDP数据**（2019-2023年）
- 数据来源：国家统计局公开数据
- 涵盖全国31个省级行政区
- 包含GDP总量、增速等多维度指标

### 2. 技术性
- **GUI框架**: Tkinter（Python标准库）
- **可视化库**: Matplotlib（专业绘图库）
- **数据库**: SQLite3（轻量级关系型数据库）
- **数据处理**: NumPy、CSV、JSON

### 3. 工程性
- 跨平台兼容（Windows/macOS/Linux）
- 中文字体自动适配
- 完整的用户文档
- 数据导入导出功能
- 图表保存功能

## 功能特点

### 可视化图表类型
1. **柱状图** - 省份GDP排名对比
2. **折线图** - 省份历年GDP趋势
3. **饼图** - GDP占比分布
4. **区域对比图** - 五大区域GDP对比

### 交互功能
- 年份选择（2019-2023）
- 省份选择（31个省份）
- 显示数量调节（Top 5-31）
- 实时统计信息更新
- 数据表格展示

### 数据管理
- 数据导出（CSV格式）
- 数据导出（JSON格式）
- 图表保存（PNG/PDF/JPG）
- SQLite数据库存储

## 安装说明

### 环境要求
- Python 3.7+
- 支持中文显示的操作系统

### 依赖安装
```bash
pip install matplotlib numpy
```

### 运行程序
```bash
python gdp_visualization.py
```

## 使用说明

### 启动程序
运行命令后，系统将自动：
1. 创建SQLite数据库
2. 导入真实GDP数据
3. 显示主界面

### 基本操作
1. **切换图表类型**: 点击左侧"图表类型"选项
2. **选择年份**: 在"年份选择"下拉框中选择
3. **选择省份**: 在"省份选择"下拉框中选择（用于折线图）
4. **调节数量**: 拖动"显示数量"滑块

### 数据导出
- **文件菜单** → **导出数据(CSV)**：导出为CSV文件
- **文件菜单** → **导出数据(JSON)**：导出为JSON文件

### 保存图表
点击"保存图表"按钮，选择保存格式（PNG/PDF/JPG）

## 数据库设计

### 表结构

#### provinces（省份信息表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| code | TEXT | 省份代码 |
| name | TEXT | 省份名称 |
| region | TEXT | 所属区域 |

#### gdp_data（GDP数据表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| province_code | TEXT | 省份代码（外键） |
| year | INTEGER | 年份 |
| gdp | REAL | GDP总量（亿元） |
| gdp_growth_rate | REAL | GDP增速（%） |
